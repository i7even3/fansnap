Fansnap API

Welcome to the Fansnap prototype server. This server exposes a JSON API for user authentication, profiles, posts, subscriptions, store items, messaging, tipping, referrals and admin functions. It does not include a fully built front‑end UI.

You can use a REST client such as Postman or curl to interact with the API.

Example Endpoints

  • POST /api/register – Register a new user
  • POST /api/login – Log in
  • GET /api/profile – View your profile
  • POST /api/posts – Create a post (creator only)
  • GET /api/posts/<creator> – View a creator's posts
  • POST /api/subscribe/<creator> – Subscribe to a creator
  • POST /api/store/<creator> – Add an item to the store (creator only)
  • POST /api/order/<itemId> – Place an order
  • POST /api/message/send – Send a direct message
  • POST /api/tip/<creator> – Send a tip

For a complete list of endpoints, view the server.js file in this project.

This is an early prototype. In a production deployment, you would build a front‑end application (e.g. React, Next.js) to consume these endpoints.
